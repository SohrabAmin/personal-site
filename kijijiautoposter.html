Kijiji Auto Poster

Python
Automation

Summary

KijijiAutoPoster is an automation tool using selenium with Python to make posting on Kijiji easer. It helps user achieve a higher selling success rate by consistently reposting ads.

Motivation

I initially thought of this project when I was trying to sell my laptop on Kijiji and found myself consistently doing the same over and over again. After freshly learning selenium, I thought that if I could automate this action of reposting it'll help me boost my ads presence.
