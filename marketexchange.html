Market Exchange

Python
Automation

Summary

Market Exchange was a school project for a software design course at UTM. The goal of the project was to basically recreate a Kijiji like program in Java while using modern software design principles. With a heavy emphases on Robert Martin's Clean Architecture along with his famous S.O.L.I.D principles. We successfully created this project from scratch with a fully working final product.

Results

Market Exchange uses a super simple command line interface, where users can create an account, post items on a virtual market and contact other users to purchase or trade them. The project really showed me the value of having a good team behind you and using the right team management tools.
